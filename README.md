# CSCI3321_Individual_Homework_2
DormPedia - College Recipe Manager

A wonderful little recipe management doohicky, designed for college students who don't know what the fuck they're doing and have minimal resources.

This was developed with heavy use of Claude.ai for debugging and syntax assistance. Certain sections (filler code and css file) were 
generated through the prompts of Sydney Cox.

All code not having a comment saying otherwise was coded by Sydney Cox.

To Start the website on localhost:
1. Run npm install to generate node_modules files, then npm install dotenv and mysql2 if needed.
2. Construct database using init.sql script and ensure schema is running on server of choice.
3. Populate a .env file with information on sql server host.
    DATABASE_HOST=
    DATABASE_USER=
    DATABASE_PASSWORD=
    DATABASE_NAME=dormpediadb
4. Call "node server.js" in project folder.
